Hola Mundo - Elixir

Overview
--------
This is a simple Elixir program that prints:

Hello, World!

Requirements
------------
- Elixir installed (version 1.14 or newer recommended)
- Erlang/OTP installed (required by Elixir)
- Windows, macOS, or Linux
- Command line terminal (CMD, PowerShell, or Bash)
- Optional: Visual Studio Code for editing

How to Run
----------
1. Create a file named hello.ex
2. Paste the following code inside the file:

   IO.puts("Hello, World!")

3. Open a terminal in the same directory.
4. Run the program with:

   elixir hello.ex

Expected Output
---------------
Hello, World!

Project Structure (Optional - Using Mix)
----------------------------------------
If using the Elixir build tool (Mix):

1. Run:
   mix new hello_project
2. Move into the folder:
   cd hello_project
3. Place your module inside the lib/ directory.
4. Run with:
   mix run

Notes
-----
- The file extension .ex is used for compiled Elixir source files.
- You can also use .exs for scripts.
- Only one line of code is required for this example.
